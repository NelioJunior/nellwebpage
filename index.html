Front End

Desenvolvemos soluções digitais de alto nível na Nelltek. Com expertise em React, Angular e PWA, criamos aplicativos mobile e web que transcendem padrões. Designs elegantes e responsivos.             

Back End

Utilizamos as linguagens Python e PHP para criar aplicativos SaaS, aplicações baseadas em micro-serviços escaláveis. Tudo em conjunto com bancos de dados robustos, como MySQL, SQL Server, PostgreSQL e MongoDB.  

Inteligencia Artificial

Utilizamos Python e bibliotecas como TensorFlow e Keras para criar modelos de IA de alta performance. Trabalhamos com APIs para modelos GPT e Lhama, permitindo a integração de conversações naturais em seus projetos.
